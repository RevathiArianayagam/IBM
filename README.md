LMS – MERN Stack Project

This is a Learning Management System (LMS) built using the MERN stack.

 Tech Stack
- MongoDB
- Express.js
- React (Vite)
- Node.js

Project Structure
- backend/ – API, models, routes, controllers
- frontend/ – React UI

Features
- User authentication
- CRUD operations
- REST API integration

 How to RunBackend
cd backend
npm install
npm run dev

Frontend
cd frontend
npm install
npm run dev
